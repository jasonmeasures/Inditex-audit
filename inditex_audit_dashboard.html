Sources & Verification

How to trust this audit. The Chapter 99 stack the engine expects is derived from the global rules in Chapter 99 Rules above. Each rule cites the executive order, the HTS code in the official tariff schedule, and its effective date window. The cards below explain every tariff layer, including how the EU cap logic resolves between 9903.02.19 and 9903.02.20. If a rule is wrong or stale, edit it above and re-run saved audits — the math is purely a function of those rules + filed MFN rates.
⚠ IEEPA & Reciprocal post-Feb 24, 2026. The Supreme Court struck down IEEPA tariffs on Feb 20, 2026 and CBP ceased collection on Feb 24, 2026. Section 122 (10%, codes 9903.03.01–9903.03.11) replaced IEEPA Universal through July 24, 2026. Entries filed before Feb 24, 2026 were assessed lawfully but are now refundable via the CBP CAPE tool (opened Apr 20, 2026).

EU Cap Logic — Why ES/PT Items Resolve to Either 9903.02.19 or 9903.02.20

AuthorityEO 14326 § 2(b) (Aug 5, 2025) MechanismFor an EU good, if the Column 1 (MFN) rate ≥ 15%, the additional ad valorem duty is 0% (code 9903.02.19). If MFN < 15%, the additional duty is set so the total reaches 15% — applied as an additive 15% via code 9903.02.20. EffectEU shipments pay a flat 15% landed reciprocal rate, never more.

China Stack — Why CN Has Three Layers

Layer 1 (Section 301)One of four mutually exclusive codes — each HTS is on exactly one list. Not affected by SCOTUS ruling — Section 301 is independently authorized under Trade Act of 1974 § 301.
 • 9903.88.01 List 1 +25% · 9903.88.02 List 2 +25% · 9903.88.03 List 3 +25% · 9903.88.15 List 4A +7.5% Layer 29903.01.24 — IEEPA China/HK fentanyl (+10%). Layer 39903.01.25 — IEEPA Universal baseline (+10%). Stack ruleAll three layers apply additively to the same entered value base — no compounding.
